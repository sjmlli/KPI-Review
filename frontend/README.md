# HRMS Frontend

React frontend for the Human Resource Management System.

## Features

- Employee Management
- Leave Management
- Attendance Tracking
- Payroll Management
- Performance Reviews
- Recruitment Management
- JWT Authentication
- Responsive Design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The app will run on `http://localhost:3000`

## Environment Variables

Create a `.env` file in the frontend directory:
```
REACT_APP_API_URL=http://localhost:8000/api/v1
```

## Technology Stack

- React 19
- React Router DOM
- Axios
- React Toastify
- Tailwind CSS
- Date-fns
